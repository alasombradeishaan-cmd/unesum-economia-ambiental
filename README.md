# unesum-economia-ambiental
La Plataforma Educativa de Economía Ambiental PI 2026 es un espacio virtual interactivo diseñado para fortalecer el aprendizaje de los estudiantes mediante teoría organizada, actividades dinámicas, juegos educativos, recursos digitales y herramientas tecnológicas que promueven la sostenibilidad, el pensamiento crítico y el aprendizaje autónomo.
